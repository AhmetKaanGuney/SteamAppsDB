API DOCUMENTATION

GET /GetAppList : returns -> list[dict]:
parameters:
    *tags         : [comma seperated tag ids] or None,
    *genres       : [comma seperated genre ids] or None,
    *categories   : [comma seperated category ids] or None,
    *order        : [comma seperated column_name,direction] or None,
    *release_date : [comparison-operator, value] || None,
    *coming_soon  : 0, 1 or None
    *index        : int
    *limit        : int
(Note: * means optional)
