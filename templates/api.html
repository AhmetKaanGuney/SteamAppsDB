API DOCUMENTATION

GET /GetAppList : returns -> list[dict]:
parameters:
    *tags       : [comma seperated tag ids] or None,
    *genres     : [comma seperated genre ids] or None,
    *categories : [comma seperated category ids] or None,
    *order      : [comma seperated column_name,direction] or None,
    *coming_soon: 0, 1 or None
    *index      : int
    *limit      : int
(Note: * means optional)

example :

Tags Example :
[
    {
        id    : 10,
        name  : "example",
        votes : 300
    },
]
Genres And Categories Example:
[
    {id: name},
]

Interfaces

AppSnippet():
    app_id  : int
    name    : str
    price   : int

    release_date: str
    coming_soon : bool

    positive_reviews: int
    negative_reviews: int
    owner_count     : int

    header_image: str

    windows : bool
    mac     : bool
    linux   : bool

    tags        : list[dict]

    categories  : list[dict]
